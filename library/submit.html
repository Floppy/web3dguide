If you want to submit a piece of code to the VRML developers library, fill in the form below. Enter your name and email address; both of these are compulsory. Submissions without a name or email address can not be entered into the repository.

Submissions of all kinds will be accepted, from complete VRML worlds to single lines of Java code. As long as it's related to VRML and will be useful to content developers, it can go in. As the colection is divided into sections, you can select which sections your code will be linked into. Don't pick too many, though, or some will be removed!

Then, fill out the boxes below. The first thing to enter is the code itself. Also, we need a short description of the code and a detailed one. The short description will be used so that people can find what they want, so make it descriptive but only a single sentence. The detailed description should contain everything that people will need to know about the code, and along with the code itself will form the web page for your submission. It should explain what your code does, and how it does it. Feel free to include URLs for examples, downloads, screenshots, anything you want.

By submitting code to the library, you agree to let the Web3D Guide distribute and publish it. If there are any copyright issues, they should be stated in a WorldInfo node in the code. However, all code submitted to the library must be free for other people to use as they see fit, without any restrictions.
